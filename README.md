# Chatbot com Python e Groq

Meu primeiro mini projeto em Python utilizando uma API de inteligência
artificial para criar um chatbot executado pelo terminal.

## Funcionalidades

- Conversa pelo terminal
- Histórico de mensagens durante a execução
- Respostas exibidas em tempo real
- Chave da API protegida por variável de ambiente
- Cores diferentes para usuário, bot e mensagens do sistema

## Tecnologias utilizadas

- Python
- Groq API
- python-dotenv
- Colorama
